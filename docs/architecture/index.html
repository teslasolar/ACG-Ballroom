Query

URL params

  • ?diagram= id (e.g. 02-presence)
  • &overlay= id (nests inside)
  • &all=1 renders every diagram

Drill-down

The system overview 00-system uses mermaid click directives. Clicking a node in the rendered SVG navigates to its sub-diagram via ?diagram=.

Build

Edit diagrams/*.mmd then run:

node docs/architecture/build.mjs

Outputs to svg/<name>.svg; this page loads them via <object> so click handlers in the mermaid SVG keep working.
